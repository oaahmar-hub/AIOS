## WHAT SHIPPED THIS WEEK (evidence: merged PRs #4–#15)

### 1. The brain — reconnected, honest, and knowledgeable
- **Reconnect the real brain** (`46aa9b1`) — wired `omar_personality_engine` into the live reply path. Replies are generated by AIOS's own personality layer, not a passthrough.
- **Hard no-fabrication rule** (`c185b57`) — the first live reply invented a listing (800 sqft / 850k that don't exist). The system prompt now forbids stating any specific unit, price, size, availability, owner, or link not present in context; the bot says it will *check and confirm* in Omar's natural style instead.
- **Knowledge connection — quotes REAL inventory** (`f8483bd`) — `inventory_retrieval.py` parses area / building / bedrooms / budget from the customer message (English **and** Arabic, incl. Arabic-Indic digits and «غرفة وصالة», ألف/مليون), searches the bundled resolver index, and injects a **VERIFIED** context block. Only genuinely quotable rows are used (real area + building + unit + price-or-size); junk areas and column-misaligned rows are filtered; sizes are quoted **without** a unit because the source mixes sqm/sqft. **237 quotable rows** available today. No anchor in the message → no results → the no-fabrication rule keeps the reply at "I'll check."

### 2. WhatsApp delivery — the loop is closed
- **Send auto-replies** (`6d144ad`) — the hosted runtime now actually delivers replies to customers.
- **Graceful fallback** (`67d8f54`) — a failing brain never means silence; the customer always gets a response.
- **Dedupe** (`7a201bb`) — replies are deduplicated by `message_id` behind an actionable gate, ending duplicate-message incidents.
- **Wasender webhook signature** (`1350d06`) accepted on the hosted endpoint.

### 3. Observability — silent failure is over
- **`/api/health/deep`** (`48c6a61`) — public, read-only, sends nothing to customers. Checks every link the live reply depends on: `runtime, resolver_db, webhook_auth, reply_mode, wasender_send, brain_n8n_openai` (a real n8n ping that catches a dead OpenAI key instantly), and `fallback_reply`. Returns `healthy | degraded | down` plus `reply_chain_live` (will a real customer message actually get answered) and a plain-English list of what's broken.

### 4. Security
- **Dashboard no longer public** (`8cc832b`) — stopped serving every `.html` publicly; the dashboard now requires auth.

### 5. Data / Truth Bridge quality
- **Authorized Scorpion inventory feed** (`63565b0`) — 140 de-duplicated units appended with source provenance; idempotent content-hash IDs. Building-unit anchors **7 → 17**.
- **JVC building-name recovery** (`f43bef6`) and **anchor repair** (`d2a52bc`) from corrupted/packed inventory strings.
- **Live-portal check verdicts** fed back into candidate bridges (`5eb29d9`).
- **Unusable-rows audit** (`da2358b`) — documented honestly: zero in-corpus recovery paths for those rows.
