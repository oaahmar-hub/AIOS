# Sales Knowledge Playbook

Lead handling, follow-up, CRM, and qualification routines.

## Extracted Knowledge
1. **Lead flow** — Lead -> CRM -> task -> property match -> dashboard is the core money workflow. (Source: CRM and AIOS planning)
2. **Qualification fields** — Capture budget, area, property type, bedroom count, and timeline early. (Source: Client manager workflows)
3. **Follow-up discipline** — Never let a lead go cold; every inquiry needs a next step or a reminder. (Source: Client manager workflows)
4. **Reply script style** — Use short premium replies that acknowledge the request and move to the next step quickly. (Source: WhatsApp canon)
5. **Task creation** — When a lead is captured, create the follow-up task immediately. (Source: CRM design)
6. **Commission awareness** — Track commissions by transaction type to keep business performance visible. (Source: Finance agent)
