# PWA to Native Plan

Fastest route:
1. Release web platform.
2. Enable PWA install.
3. Validate iPhone Safari install.
4. Validate Android Chrome install.
5. Wrap PWA with Capacitor only after beta is stable.

Native package identifiers:
- iOS: `com.hshglobaldubai.aios`
- Android: `com.hshglobaldubai.aios`

Required before store submission:
- Apple Developer account access
- Google Play Console access
- App icon PNG set
- Screenshots
- Privacy policy URL
- Support URL
- Test account credentials if app review requires login
- Data safety questionnaire
