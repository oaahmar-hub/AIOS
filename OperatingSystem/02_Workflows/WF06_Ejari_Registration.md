# WF06 — Ejari Registration

**Flow:** Tenancy signed → docs → register → certificate → CRM
**Trigger:** A tenancy contract is signed (must register within **30 days**, `LEGAL_KNOWLEDGE.md` #4).
**Lead agent:** [A04 Compliance Officer](../01_Agents/A04_Compliance_Officer.md) · **Supports:** A06, A07

## Steps
1. **Deadline clock** — set the 30-day registration deadline from the signing date; create the task immediately.
2. **Documents** ([SOP03](../03_SOPs/SOP03_Ejari_Registration.md)): signed tenancy contract, title deed copy, landlord passport/EID, tenant passport/EID/visa, DEWA premise number, prior Ejari (if renewal).
3. **Verify the contract** terms with [A06](../01_Agents/A06_Contracts_Renewals.md) before registering.
4. **Register** via Ejari/DLD channel (typist/portal).
5. 🔒 **Fees** — registration fee payment → Omar/typist executes.
6. **Certificate** — obtain the Ejari certificate; file to Drive; attach to the unit/tenancy record.
7. **CRM/register** — mark registered with the Ejari number; close the deadline task.

## Outputs
- Ejari certificate + number on file, deadline task closed, tenancy record updated.

## Done when
- Registered within 30 days, certificate filed, CRM updated.
