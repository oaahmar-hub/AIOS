# SOP08 — AML / KYC Compliance

**Purpose:** Meet UAE AML/KYC obligations on every transaction and escalate suspicious activity correctly.
**When:** Every sale/transfer; any cash-heavy or unusual transaction.
**Owner:** [A04](../01_Agents/A04_Compliance_Officer.md)

## Prerequisites / Documents
- Client ID (passport/Emirates ID), proof of address
- Source-of-funds evidence for the transaction
- Beneficial-owner identification (for entities)
- Sanctions/PEP screening result

## Steps
1. **KYC** — collect and verify client identity + beneficial owner before proceeding.
2. **Source of funds** — obtain and document evidence appropriate to the deal size.
3. **Screen** against sanctions/PEP lists; record the result.
4. **Risk-rate** the client/transaction (low/medium/high); apply enhanced due diligence for high.
5. **Record-keeping** — retain KYC + transaction records for **5 years** (`LEGAL_KNOWLEDGE.md` #7).
6. 🔒 **Suspicious transaction** — escalate to Omar; file via **goAML** within the required window (`LEGAL_KNOWLEDGE.md` #8). Do not tip off the client.

## Fees & Deadlines
- **5-year** record retention
- goAML suspicious-transaction filing within the mandated window

## Outputs
- Completed KYC file, source-of-funds evidence, screening result, risk rating, retained records; STR filed if applicable.

## Risk-Hold points
- 🔒 Any suspicious-transaction concern → Omar + goAML, never client-facing advice.
