# SOP03 — Ejari Registration

**Purpose:** Register a tenancy contract in Ejari within the legal window.
**When:** Any tenancy contract is signed (new or renewal).
**Owner:** [A04](../01_Agents/A04_Compliance_Officer.md) · **Workflow:** [WF06](../02_Workflows/WF06_Ejari_Registration.md)

## Prerequisites / Documents
- Signed tenancy contract (all pages)
- Title deed copy
- Landlord passport / Emirates ID (or POA)
- Tenant passport / Emirates ID / visa
- DEWA premise number (or developer/affection plan reference)
- Previous Ejari certificate (for renewals)

## Steps
1. Set the **30-day** deadline clock from the signing date; create the task immediately.
2. Verify contract terms ([A06](../01_Agents/A06_Contracts_Renewals.md)) before registering.
3. Compile the document set above.
4. Register via the Ejari/DLD channel (approved typing centre or portal).
5. 🔒 Pay the registration fee (Omar/typist executes).
6. Obtain the Ejari certificate + number; file to Drive; attach to the tenancy/unit record.
7. Close the deadline task; mark registered in CRM.

## Fees & Deadlines
- **Register within 30 days of signing** (`LEGAL_KNOWLEDGE.md` #4)
- Ejari registration fee + typing/admin (confirm current)

## Outputs
- Ejari certificate + number filed; deadline task closed; record updated.

## Risk-Hold points
- 🔒 Fee payment.
