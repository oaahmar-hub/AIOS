# SOP04 — RERA Advertising Permit (Trakheesi)

**Purpose:** Obtain the advertising permit required before publishing any paid property listing.
**When:** Before publishing/boosting any listing on Property Finder, Bayut, Dubizzle, or paid social.
**Owner:** [A04](../01_Agents/A04_Compliance_Officer.md) / [A03](../01_Agents/A03_Listings_Manager.md) · **Workflow:** [WF03](../02_Workflows/WF03_PF_Listing_Publish.md)

## Prerequisites / Documents
- Valid brokerage trade licence + RERA broker registration
- Signed Form A (seller/landlord listing agreement) for the unit
- Title deed copy
- Owner ID
- Unit details matching the listing

## Steps
1. Confirm a signed Form A authorises HSH to advertise the unit.
2. Apply for the Trakheesi advertising permit via the DLD/RERA channel.
3. Receive the permit number.
4. Place the permit number on the listing **before** it goes live.
5. Track permit validity; renew before expiry; do not let a listing run on an expired permit.

## Fees & Deadlines
- Permit fee per advertisement/period (confirm current)
- **A RERA permit is required before publishing paid listings** (`LEGAL_KNOWLEDGE.md` #3)

## Outputs
- Permit number recorded against the listing in the pipeline tracker.

## Risk-Hold points
- 🔒 Publishing any paid listing without a valid permit number → hard stop.
